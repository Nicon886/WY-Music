# 仿网易云音乐小程序

一直都是在酷狗和网易云上听音乐，对于这两个音乐平台，个人更喜欢网易云的，其布局也是一直深受个人喜爱。想着就拿了网易云音乐来进行微信小程序上的复刻。

## 网易云音乐API接口

音乐当然少不了必需的音乐API接口了，这里用到的网易云API非常的棒，很全面，喜欢的可以到原作者那支持一下。地址戳这里-[网易云音乐API](https://binaryify.github.io/NeteaseCloudMusicApi/#/)
